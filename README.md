# Node SMS Text App

Node.js app that uses Nexmo to send SMS text messages

### Version
1.0.0

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser

```sh
$ npm start
```

## App Info

### Nexmo API Keys
You will need to add your own Nexmo api key, secret and virtual number
